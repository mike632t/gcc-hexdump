## gcc-hexdump - Dump files in hex and convert to/from binary.

Written in C.   

Three programs:

gcc-hexdump.c
   
   Prints  the contents of one or more files in hex and (optionally) octal.
   
   
gcc-unload.c
   
   Prints  the  contents of one or more files in eight bit Intel Hex  format
   (the  same as the output produced by the  CP/M-80  'unload'  command), or
   Motorola 'S' format.
   
   
gcc-load.c
   
   Reads  an eight bit Intel Hex or Motorola 'S' format file and writes the
   records to a binary file.
